Work Description

  • Software Developer/Engineer:
    • Write, test, and maintain code for software applications or systems.
    • Collaborate with cross-functional teams to design and develop software.
    • Debug and troubleshoot issues in code
  • Data Scientist/Analyst:
    • Write, test, and maintain code for software applications or systems.
    • Collaborate with cross-functional teams to design and develop software.
    • Debug and troubleshoot issues in code
  • Web Developer::
    • Create and maintain websites and web applications.
    • Develop user interfaces, layouts, and responsive designs.
    • Ensure web security and optimize site performance.
  • System Administrator:
    • install and configure operating systems and software on servers and computers.
    • Perform system maintenance and updates.
    • Provide technical support to end-users.
  • IT Manager/Director:
    • Oversee IT projects, budgets, and staff.
    • Make strategic decisions regarding technology infrastructure.
    • Ensure technology aligns with business goals.
